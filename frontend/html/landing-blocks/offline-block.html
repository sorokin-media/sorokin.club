Оффлайн

Клуб - это не только сайт и чаты. Самое интересное происходит в оффлайне.

  • Регулярные встречи в разных городах.
  • Голосовые конференции для обсуждения наболевших вопросов.
  • Тусовки для приятного активного времяпрепровождения: небольшие походы, прогулки, заплывы на яхтах и прочее.
  • Совместные путешествия.
  • Большие закрытые клубные конференции.
